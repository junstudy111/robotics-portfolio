# 의류 정리 자동화 시스템

비정형 의류 파지를 고려한 로봇 자동화 프로토타입

## 📌 프로젝트 개요
- **기간:** 2024.11.10 ~ 11.21
- **팀:** 4명
- **역할:** 로봇 제어 & Auto Recovery

## 🛠 Tech Stack
- Python
- ROS
- Firebase
- Doosan M0609 Manipulator

## 📁 Files
- `robot_control.py` - 로봇 제어 메인 코드
- `dashboard.html` - 웹 대시보드

## ✨ 핵심 기능
- MoveJ/MoveL 동작 제어
- Step Save & Auto Recovery
- Firebase 실시간 연동
- 긴급정지 및 충돌 감지
