# 로봇 기반 공공시설 심정지 초동대응 시스템

지하철 심정지 환자 발생 시 AED 이송 및 구조대 안내 로봇

## 📌 프로젝트 개요
- **기간:** 2024.12.29 ~ 2025.01.06
- **팀:** 7명
- **역할:** Robot B 제어 (구조대 안내 로봇)

## 🎬 Demo Video
[YouTube 링크](유튜브링크)

## 🛠 Tech Stack
- Python
- ROS2 Jazzy
- Nav2
- TurtleBot4
- YOLO

## 📁 Files
- `robot_b.py` - Robot B 제어 메인 코드 (구조대 안내)
- `oakd_yolo.py` - YOLO 객체 인식

## ✨ 핵심 기여 (Robot B)

### 1. 이벤트 기반 상태 머신 설계
- 콜백 체인 방식 동작
- 플래그 기반 상태 관리
- 긴급 중단 시 모든 타이머 취소

### 2. Nav2 자율주행 제어
- 다중 목표 순차 이동
- 실시간 경로 계획
- 장애물 회피

### 3. 안전 제어 로직
- 군중 감지 자동 정지/재개
- 긴급 중단 명령 처리
- 비프음 경고 시스템

## 🚀 주요 기능
- 언도킹 → 1차 목표 이동 → Rescue 위치 대기
- 구조대 감지 → 환자 위치 안내
- 정지 명령 → 도킹 복귀

## 🎯 프로젝트 성과
- 단일 로봇 자율주행 시나리오 완벽 구현
- 이벤트 기반 상태 머신 안정적 동작
- 실시간 안전 제어 로직 정상 작동

## 💡 배운 점
